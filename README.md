# Covid-19-App

## Features:
1. Able to show the details of Covid-19 by each Coounties.
2. Mobile responsive.
3. Able to show the data by daily till today.
4. Show us the static data by Graph chart and Bar chart. 

Live demo: https://tinder-clone-2b0ce.web.app/
